# Fullstack Intro Lab

## 📁 Project Structure

This project demonstrates a basic client-server architecture.

---

## 🔄 Client-Server Architecture (REAL FLOW)

1. User opens frontend (browser)
2. Frontend sends request using fetch/axios
3. Backend receives request (Express server)
4. Backend processes data
5. Backend sends response (JSON/text)
6. Frontend updates UI

---

## 📌 Example Request Flow

Frontend Button Click → API Request → Backend Route → Response → UI Update

---

## 🛠 Technologies Used

- HTML / React (Frontend)
- Node.js + Express (Backend)
- VS Code (Development IDE)

---

## 💡 Key Concept Demonstrated

This project shows **unidirectional data flow between client and server**.
